# NanTube
Аналог ютуба(работает по локальной сети и со всех устройств) для подготовки к чебурнету

# Как запустить
- **Проверьте, установлен ли у вас пайтон, если нет установите, без этого не работает**
- Откройте терминат и введите cd и путь в папку в которой находится requirements.txt
- Потом введите **pip install -r requirements.txt**
- Откройте app.py и найдите ```python VIDEO_FOLDER = r'E:\videos'  # Измените на абсолютный путь к папке с видео ```
- Запускаете app.py
